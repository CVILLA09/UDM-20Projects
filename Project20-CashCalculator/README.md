<!-- Features -->

### Key Features <a name="-key-features"></a>

- **User-friendly Interface:** Provides input fields for different dollar bill denominations ($100, $50, $20, $10, $5, $2, $1).
- **Visual Representation:** Displays images of each bill alongside input fields with corresponding numeric values.
- **Reset Functionality:** Includes a "Reset" button to clear all input values.
- **Real-time Updates:** Shows the total cash amount and its equivalent in words dynamically.
